Workshops

I have created and helped facilitate many workshops on applied mathematics, data analysis, and programming. Below are some of the workshops and materials.

PI4 Applied Linear Algebra Working Group

An intensive multi-day workshop I created on advanced linear algebra with applications in statistics and data analysis.

Linear algebra topics included orthogonalization, matrix factorizations, spectral theorem, dual spaces, and an introduction to symplectic linear algebra.

Appllications included least-squares optimization, linear models and regression, polynomial interpolation, trend analysis, principal components analysis, and dimensional reduction.

The workshop consisted of guided active-learning sessions and mini-lectures.

Some of the materials used:

  • Worksheet 1: Gram-Schmidt orthogonalization, QR factorization, least-squares optimization, linear models and regression, polynomial interpolation, trend analysis
  • Worksheet 2: diagonalization, spectral theorem, singular value decomposition (SVD)
  • Worksheet 3: Principal component analysis, dimensional reduction
  • Worksheet 4: dual spaces, annihilators, symplectic linear algebra

Created with support from National Science Foundation grant DMS 1345032 “MCTP: PI4: Program for Interdisciplinary and Industrial Internships at Illinois.”

Originally developed to accompany another computational bootcamp helping graduate students participating in internships.

Illinois Geometry Lab Computational Workshop

A workshop I created introducing undergraduates to Python scientific computing. Aimed at students that would go on to work on undergraduate research projects during the semester.

The workshop consisted of lectures and active-learning sessions. Jupyter notebooks were designed for students to follow along to lectures before the activities.

Workshop included introductions numpy, scipy, and matplotlib.

  • Notebook 1: Introduction to Python
  • Notebook 2: Matplotlib Basics
  • Notebook 3: Solving equations numerically with Python
  • Notebook 4: Solving differential equations, dynamical systems, and more with Python

PI4 Computational Bootcamp on Data Analysis

A workshop I helped facilitate introducing mathematics students to statistics, data analysis, and statistical learning using both Python and R.

The workshop included interactive lectures and small data analysis research projects.

Topics covered included data wrangling and visualization with both R and Python, data mining, statistical modeling, clustering algorithms, and an introduction to cloud computing for big data analytics. Students were also introduced to version control and end-to-end data analysis workflows.

Joint with David LeBauer and Neil Davis.

PI4 Computational Bootcamp on Scientific Computing

A workshop I helped facilitate on numerical computation for math students going on to participate in internships.

Topics covered introductions to working with Python for scientific computing. Students worked through interactive Jupyter notebooks introducing numpy, scipy, and matplotlib.

Workshop lead: Anil Hirani.
